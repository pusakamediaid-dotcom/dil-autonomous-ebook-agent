# STYLE GUIDE
# DIL Autonomous Ebook Agent
# Versi: 1.0

## Bahasa

- Gunakan Bahasa Indonesia yang formal tetapi mudah dipahami.
- Hindari terjemahan harfiah istilah teknis yang tidak lazim.
- Gunakan istilah teknis sesuai TERMINOLOGY_RULES.md.

## Format Markdown

- Heading bab menggunakan ##.
- Heading subbab menggunakan ###.
- Heading sub-subbab menggunakan ####.
- Jangan melompat level heading.
- Gunakan teks tebal untuk istilah penting pertama kali muncul.
- Gunakan format kode untuk nama variabel, fungsi, dan perintah.

## Rumus Matematika

- Rumus inline menggunakan tanda dollar tunggal.
- Rumus display menggunakan tanda dollar ganda.
- Selalu sertakan penjelasan setiap simbol setelah rumus.

## Tabel

- Selalu sertakan header row.
- Gunakan alignment yang konsisten.
- Lebar tabel tidak boleh melebihi 100 karakter.

## Callout

- Gunakan danger untuk peringatan keselamatan.
- Gunakan tip untuk tips berguna.
- Gunakan note untuk catatan tambahan.

## Panjang Konten

- Setiap subbab minimum 400 kata.
- Setiap bab minimum 800 kata.
- Setiap contoh perhitungan minimum 5 langkah.
