# PROJECT BIBLE
# DIL Autonomous Ebook Agent
# Versi 1.0

## Identitas Proyek

Nama proyek: DIL Autonomous Ebook Agent.
Pemilik: DIL / pusaka.media.id.
Tujuan: membantu produksi ebook teknis premium secara bertahap, terstruktur, aman, dan mudah diawasi.
Platform: GitHub Actions, Python, Markdown, Pandoc, dan LaTeX.

## Tujuan Utama

Sistem ini dibuat untuk membantu DIL menghasilkan draft ebook teknis yang rapi, konsisten, dan siap direview sebelum dijadikan produk final.

## Standar Kualitas Minimum

- Setiap subbab memakai Metode 5-Lapis.
- Skor review minimum sebelum PDF final adalah 70 dari 100.
- Rumus harus memakai sintaks LaTeX yang benar.
- Istilah teknis harus konsisten.
- Konten harus aman untuk pembaca pemula.

## Metode 5-Lapis

1. [KONSEP]
2. [ANALOGI]
3. [RUMUS]
4. [CONTOH]
5. [APLIKASI]

## Struktur Ebook Standar

- Front Matter: sampul, disclaimer, kata pengantar, daftar isi.
- Body: bab dan subbab utama.
- Back Matter: soal latihan, glosarium, dan referensi.

## Aturan Operasional

- Gunakan mode test sebelum mode besar.
- Gunakan batas biaya dan batas token.
- Hentikan proses jika batas terlampaui.
- Semua laporan harus disimpan agar proses bisa diaudit.
