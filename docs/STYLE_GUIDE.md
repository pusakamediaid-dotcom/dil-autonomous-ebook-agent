# DIL Ebook Agent - Style Guide

## Writing Style

### Bahasa
- Gunakan Bahasa Indonesia untuk konten ebook
- Gunakan Inggris untuk kode dan teknis istilah
- Formal tapi mudah dipahami

### Struktur Konten

Setiap subsection HARUS memiliki:

1. **[KONSEP]** - Penjelasan konseptual
   - Definisi jelas
   - Prinsip dasar
   - Konteks penggunaan

2. **[ANALOGI]** - Perumpamaan untuk pemahaman
   - Hubungkan dengan hal familiar
   - Sederhana dan relatable

3. **[CONTOH]** - Contoh praktis
   - Kode yang bisa dijalankan
   - Step-by-step

4. **[APLIKASI]** - Penerapan nyata
   - Use cases
   - Best practices

Opsional:
5. **[RUMUS]** - Formula atau prinsip matematis (jika applicable)

### Heading Format

```
# Judul Ebook
## Chapter X: Judul Bab
### X.Y Section Title
#### X.Y.Z Subsection Title
```

### Code Blocks

```python
# Always specify language
def example():
    pass
```

### Emphasis

- **Bold**: Konsep penting
- *Italic*: Istilah teknis
- `Code`: Kode atau command

## Template Subsection

```markdown
### [KONSEP]

Penjelasan konsep dengan bahasa yang jelas...

### [ANALOGI]

Analoginya...

### [CONTOH]

Contoh kode atau implementasi...

### [APLIKASI]

Bagaimana mengaplikasikannya...
```

## Checklist Kualitas

- [ ] Setiap subsection punya KONSEP, ANALOGI, CONTOH, APLIKASI
- [ ] Kode bisa dijalankan
- [ ] Tidak ada placeholder seperti "Lorem ipsum"
- [ ] Consistent formatting
- [ ] Grammar yang benar
