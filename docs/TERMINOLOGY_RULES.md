## Usage Rules

1. **Indonesian First**: Gunakan istilah Indonesia untuk penjelasan
2. **English for Code**: Kode tetap dalam bahasa Inggris
3. **Consistent Translation**: Satu istilah = satu arti konsisten
4. **Technical Terms**: Jika tidak ada padanan, gunakan Inggris

## Forbidden Terms

- Jangan gunakan "gratis" atau "free" tanpa konteks
- Jangan gunakan "unlimited" untuk API usage
- Jangan gunakan "best" untuk model comparison

## Capitalization

- Nama produk: GPT-4, Claude 3, Gemini
- Nama provider: OpenAI, Anthropic, Google
- Nama file: snake_case.md
- Nama class: PascalCase
- Nama function: snake_case()
